# Web UI (Next.js)

This folder contains a minimal Next.js scaffold for the Simple Flight Compensation UI.

Quick start:

```bash
cd web
npm install
npm run dev
```

Pages:
- `/` — landing with links
- `/passenger` — passenger flow stub (register flight, view status)
- `/airline` — airline flow stub (view claims, accept/deny)

This is an initial scaffold with placeholder UI only; Web3 integration and contract wiring will be implemented next.
